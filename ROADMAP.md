# FarmOS Roadmap

## Phase 1” Origami Farms Internal MVP

Goal: build a working system for daily use at Origami Farms.

Scope:

- Farm Knowledge Model
- Animal Digital Twin
- Feed workflow
- Dairy workflow
- Poultry workflow
- Veterinary observations
- Inventory basics
- Morning briefing
- Offline-first tablet experience

## Phase 2” Origami Farms Optimized

Goal: improve based on real use.

Scope:

- Improved AI recommendations
- Better dashboards
- Produce management
- Sales tracking
- Profitability per animal
- More advanced reports

## Phase 3” Commercial Readiness

Goal: prepare FarmOS for use by other farms.

Scope:

- Multi-farm support
- User roles and permissions
- Setup wizard
- Configurable species
- Subscription-ready architecture
- Documentation website

## Phase 4” FarmOS Professional

Goal: support larger operations.

Scope:

- Employees
- Payroll
- Machinery
- Maintenance
- Advanced finance
- Supplier evaluation
- Advanced analytics

## Phase 5” FarmOS Platform

Goal: become an intelligent agriculture platform.

Future scope:

- IoT integrations
- RFID
- Sensors
- Weather stations
- AI vision
- Marketplace
- Veterinary tele-support
