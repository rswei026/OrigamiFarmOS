# FarmOS Glossary

This glossary defines the official vocabulary of FarmOS.

Every term must have one clear meaning.

## Core Terms

### Observation

An objective record of reality captured at a specific point in time.

### Knowledge

Validated information derived from observations and context.

### Recommendation

An explainable suggestion generated by FarmOS to support human decision-making.

### Decision

An action selected by an authorized human.

### Outcome

The measured result of a decision or action.

### Learning

The process by which outcomes improve future recommendations.

### Digital Twin

The authoritative digital representation of a real farm entity.

### Event

An immutable fact representing something that happened.

### FarmOS Intelligence

The capability of FarmOS to transform observations into evidence-based recommendations.
